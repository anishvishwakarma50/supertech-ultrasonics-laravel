## 📌 About The Company

**Super Tech Ultrasonics** is a manufacturer and exporter of high-quality ultrasonic cleaning machines designed for industrial and precision cleaning applications. The company specializes in crafting advanced ultrasonic cleaning solutions used across various industries.

### Products Include
* Industrial Ultrasonic Cleaning Systems
* Table Top Ultrasonic Cleaners
* Immersible Ultrasonic Transducers
* Ultrasonic Medical Cleaners
* Customized Ultrasonic Cleaning Solutions

## 🚀 Project Overview

This project is the official company website built using **Laravel**. It showcases company information, products, and industrial cleaning solutions offered by Super Tech Ultrasonics.

### Key Features
* **Dynamic Product Management:** Easily update and manage the product catalog.
* **Inquiry System:** Integrated lead generation for potential customers.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **SEO-Friendly:** Structured for better visibility on search engines.

## ⚙️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anishvishwakarma50/supertech-ultrasonics-laravel.git

2. **Navigate into the project directory:**
   ```bash
   cd supertech-ultrasonics-laravel

3. **Install dependencies:**
   ```bash
   composer install

4. **Copy environment file:**
   ```bash
   cp .env.example .env

5. **Generate application key:**
   ```bash
   php artisan key:generate

6. **Run Migrations:**
   ```bash
   php artisan migrate

7. **Link the Storage**
   ```bash
   php artisan link:storage

8. **Run Migrations:**
   ```bash
   php artisan migrate
9. **Start the Development serve:**
   ```bash
   php artisan serve
